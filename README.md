Building with CMake on GNU/Linux
-----------------------------------
1. Clone the repository: `https://github.com/WITopenCV/display-image`
2. Navigate to the newly cloned repository's directory: `cd display-image`
5. Run CMake: `cmake`
6. Run GNU Make: `make`
